# Usage
```cpp
IntSelect<SELECTION, Int1, Int2...>
```

# Arguments
 * SELECTION: FUNCTION
 * N: numbers
 * return value: INTEGER

# Description
Returns SELECTION of N
If SELECTION is 0, the first integer is returned, if SELECTION is 1, the second and so forth.
