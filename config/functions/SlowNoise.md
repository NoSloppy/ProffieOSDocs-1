title: SlowNoise
---

# Usage
```cpp
SlowNoise<SPEED>
```

# Arguments
return value: FUNCTION

# Description
Returns a value between 0 and 32768 which changes randomly up and
down over time. All pixels gets the same value.
SPEED controls how quickly the value changes.
