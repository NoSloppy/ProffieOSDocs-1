title: IsLessThan
---

# Usage
```cpp
IsLessThan<F, V>
```

# Arguments
 * F, V: FUNCTION
 * return value: FUNCTION

# Description
Returns 0 or 32768 based on V
If F < V returns 32768, if F >= V returns 0
