title: BrownNoiseF
---

# Usage
```cpp
BrownNoiseF<GRADE>
```

# Arguments
 * return value: FUNCTION

# Description
Returns a value between 0 and 32768 with nearby pixels being similar.
GRADE controls how similar nearby pixels are.
