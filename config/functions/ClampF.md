# Usage
```cpp
ClampF<F, MIN, MAX>
   ClampFX<F, MINCLASS, MAXCLASS>
```

# Arguments
 * F, MIN, MAX: INTEGER
 * MINCLASS, MAXCLASS: FUNCTION
 * return value: INTEGER

# Description
Clamps value between MIN and MAX
