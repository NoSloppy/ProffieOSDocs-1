title: WavNum
---

# Usage
```cpp
WavNum<>
WavNum<EFFECT>
```

# Arguments
EFFECT: effect type
return value: INTEGER

# Description

Returns which file was actually played.
First file returns 0. Even if the file is called 'clash1.wav'.
