title: BulletCount

# Usage
```cpp
BulletCount
```

# Arguments
No arguments

# Description
Return 0-N (number of bullets in the magazine.)
Will cause compilation error if your prop doesn't have bullets.
You must define BLASTER_SHOTS_UNTIL_EMPTY for this to be available.
