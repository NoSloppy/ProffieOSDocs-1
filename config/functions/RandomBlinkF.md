title: RandomBlinkF
---

# Usage
```cpp
RandomBlinkF<MILLIHZ>
```

# Arguments
 * MILLHZ: FUNCTION

# Description
Randomly returns either 0 or 32768 for each LED. The returned value
is held, but changed to a new random value MILLIHZ * 1000 times per
second.
