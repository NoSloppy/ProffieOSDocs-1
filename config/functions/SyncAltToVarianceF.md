title: SyncAltToVarianceF
---

# Usage
```cpp
SyncAltToVarianceF
```

# Arguments
 * return value: INTEGER (always zero)

# Description
Enables Bidirectional synchronization between ALT and VARIANCE.
If variance changes, so does alt, if alt changes, so does variance.
