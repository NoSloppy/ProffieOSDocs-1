# Usage
```cpp
OriginalBlastF<EFFECT>
```

# Arguments
EFFECT: a BladeEffectType (defaults to EFFECT_BLAST)
return value: FUNCTION

# Description
Original blast function. Normally returns zero, but
returns up to 32768 when the selected effect occurs.
