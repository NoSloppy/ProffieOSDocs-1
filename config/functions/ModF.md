---

# Usage
```cpp
ModF<F, MAX>
```

# Arguments
 * F: FUNCTION
 * MAX: FUNCTION (not zero)

# Description
When F is greater than MAX, F wraps to 0
When F is less than 0, F wraps to MAX
returns Integer
