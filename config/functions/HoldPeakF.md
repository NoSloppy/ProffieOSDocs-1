title: HoldPeakF
---

# Usage
```cpp
HoldPeakF<F, HOLD_MILLIS, SPEED>
```

# Arguments
F, HOLD_MILLIS and SPEED: FUNCTION
return value: FUNCTION, same for all LEDs

# Description
Holds Peak value of F for HOLD_MILLIS.
then transitions down over SPEED to current F
