# Usage
```cpp
IntSelectX&lt;SELECTION, F1, F2, ...&gt;
```

# Arguments
SELECTION: FUNCTION

# Description
Returns SELECTION of N
If SELECTION is 0, the first function is returned, if SELECTION is 1, the second and so forth.
F1, F2...: FUNCITON
return value: INTEGER
