title: RampF  
---

# Usage
```cpp
RampF  
```

# Arguments
No arguments

# Description
Returns 0 at base and 32768 at tip.
Example: Mix<RampF, COLOR1, COLOR2, COLOR3>
This would do the same thing as Gradient<COLOR1, COLOR2, COLOR3>
