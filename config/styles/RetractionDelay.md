title: RetractionDelay
---

# Usage
```cpp
RetractionDelay<DELAY_MILLIS, BASE>
```

# Arguments
DELAY_MILLIS: a number
BASE: COLOR
return value: COLOR

# Description
This class renders BASE as normal, but delays retraction by
the specified number of milliseconds.
