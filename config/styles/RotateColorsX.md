title: RotateColorsX
---

# Usage
```cpp
RotateColorsX<ROTATION, COLOR>
```

# Arguments
ROTATION: FUNCTION
COLOR: COLOR or LAYER
return value: COLOR or LAYER (same as COLOR)

# Description

ROTATION specifies how much to rotate the color in HSV (color wheel)
space. 0 = none, 32768 = 360degrees
