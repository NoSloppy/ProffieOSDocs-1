# Usage
```cpp
BrownNoiseFlicker&lt;A, B, grade&gt;
BrownNoiseFlickerL&lt;B, grade&gt;
```

# Arguments
A, B: COLOR
grade: int
return value: COLOR

# Description
Randomly selects between A and B, but keeps nearby
pixels looking similar.
