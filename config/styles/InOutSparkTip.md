# Usage
```cpp
InOutSparkTip&lt;BASE, OUT_MILLIS, IN_MILLIS, SPARK_COLOR&gt;
```

# Arguments
BASE, SPARK_COLOR: COLOR
OUT_MILLIS, IN_MILLIS: a number
return value: COLOR

# Description
Similar to InOutHelper, but makes the tip a different color
during extension.
