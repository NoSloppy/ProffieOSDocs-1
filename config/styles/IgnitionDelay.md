title: IgnitionDelay
---

# Usage
```cpp
IgnitionDelay&lt;DELAY_MILLIS, BASE&gt;
```

# Arguments
DELAY_MILLIS: a number
BASE: COLOR
return value: COLOR

# Description
This class renders BASE as normal, but delays ignition by
the specified number of milliseconds. Intended for kylo-style
quillions.
