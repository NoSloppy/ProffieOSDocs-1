# Usage
```cpp
TrSelect<SELECTION, TR1, TR2, ...>
```

# Arguments
SELECTION: FUNCTION
TR1, TR2: TRANSITION
return value: TRANSITION

# Description
transition option is picked from the specified list of
transitions based on Int<>
with Int<0> representing first transition
