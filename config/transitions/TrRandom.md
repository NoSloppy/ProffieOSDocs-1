title: TrRandom
---

# Usage
```cpp
TrRandom<TR1, TR2, ...>
```

# Arguments
TR1, TR2: TRANSITION
return value: TRANSITION

# Description
Each time a new transition is started, a random
transition is picked from the specified list of
transitions.
