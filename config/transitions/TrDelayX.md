# Usage
```cpp
TrDelayX<MILLIS_FUNCTION>
TrDelay<MILLIS>
```

# Arguments
 * MILLIS_FUNCTION: FUNCTION
 * MILLIS: a number
 * return value: TRANSITION

# Description
Waits for the specified number of milliseconds, then transitions
to second color. Meant to be used with TrConcat
