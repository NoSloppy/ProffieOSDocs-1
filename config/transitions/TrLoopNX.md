# Usage
```cpp
TrLoopNX&lt;N_FUNCTION, TRANSITION&gt;
TrLoopN&lt;N, TRANSITION&gt;
```

# Arguments
N_FUNCTION: FUNCTION (number of Loops)
N: a number (Loops)
TRANSITION: TRANSITION
Return Value: TRANSITION

# Description
Runs the specified transition N times.
