# Usage
```cpp
TrColorCycle<MILLIS, START_RPM, END_RPM>
```

# Arguments
 * OR:  TrColorCycleX<MILLIS_FUNCTION, START_RPM, END_RPM>
 * MILLS:  number
 * MILLIS_FUNCTION: FUNCTION
 * START_RPM: a number (defaults to 0)
 * END_RPM: a number (defaults to 6000)
 * return value: COLOR

# Description
Tron-like transition.
